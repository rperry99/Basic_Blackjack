# Basic Blackjack

This is mostly just a fun project for myself to make a game of blackjack. The game is played just against yourself with a dealer.

I would like this known that I don't know how to play the actual game of blackjack, and am more or less just basing this off trying to get as close to 21 without going over as possible.

## Some Features of this game

- [x] All cards are dynamically created, and are not just images. I wanted to get some practice with CSS.
- [x] A full deck of cards is created, and then shuffled using the [Fisher-Yates Shuffle](https://en.wikipedia.org/wiki/Fisher%E2%80%93Yates_shuffle).
- [x] Hitting "Deal Card" will generate a new card. This card is the top card of the deck. That card is then removed from the deck.
